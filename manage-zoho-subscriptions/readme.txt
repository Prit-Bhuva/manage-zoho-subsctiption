A custom WordPress plugin integrating Zoho API and Vonage for subscription management, calls, and SMS notifications.

== Description ==

This plugin automates subscription management using Zoho API and enables calling and SMS functionality through Vonage. It supports creating customers, managing payments, and notifying users when caring people don't respond to calls.

== Installation ==

1. Upload `manage-zoho-subscription` to the `/wp-content/plugins/` directory.
2. Activate through the 'Plugins' menu in WordPress.
3. Configure Zoho and Vonage API credentials.
